# unable to locate adb

Created: 2021년 10월 4일 오후 3:49
Tags: Android, Build

## 현상

```jsx
Unable to locate adb
```

## 원인

- 프로젝트 SDK 연결 안됨

## 대처

- File > Project Structure > Prject Settings > Project 의 Project SDK 가 no SDK 로 된 부분을 올바르게 설정한다.
